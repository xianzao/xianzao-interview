## 每日一道前端面试题

1. 出题时间：

    1. 工作日

2. 内容范围--包括但不限于以下前端基础:

    1. HTML
    2. CSS
    3. JavaScript
    4. 网络通信
    5. Vue
    6. React
    7. TypeScript
    8. Node
    9. 构建工具

3. 包含领域--包括但不限于以下热门领域:

    1. 微前端
    2. low code/no code
    3. 前端工程化搭建
    4. 自动化部署
    5. 前端性能检测
    6. 跨端
    7. 前端组件化
    8. 前端稳定性建设
